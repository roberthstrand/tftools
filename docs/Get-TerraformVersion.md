# Get-TerraformVersion

## SYNOPSIS
Display all installed versions of Terraform.

## SYNTAX

```
Get-TerraformVersion [-SilentlyRun]
```

## DESCRIPTION
Display all versions of Terraform that are in the library, highlighting the current active version.

## EXAMPLES

### EXAMPLE 1
```
Get-TerraformVersion
```
